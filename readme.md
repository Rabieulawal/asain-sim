so basically thare are your stats which include 

### 1. FP

### REB (rebilion)
this is how much you rebel with your parents if you let thsi become 100 that means that you rebeled more then your parents limit and you will be hit by a flying projectile AKA" THE SLIPPER " which will doom you to end the game 

### PAS (passion)
this is how much you follow your passion if this becomes 0 then you will become a zombie which means that you will just run on auto pilot and time will move fast and by the time you know you will already be a nurosurgon but still need 4 hours of counselling a week becouse of all the things that happaned to you in your childhood

this is your sanity every desision you make which overwhems you reduces it if it reaches 0 then the game ends

## Running from source files

Before you can run the source code for this game, you need to set up a few tools on your computer. Follow these steps to get everything installed:

### Step 1: Install Python & Pip
`pip` is the package manager for Python, and it comes automatically installed when you download Python from the official website.

1. Go to the [Official Python Website](https://www.python.org/downloads/).
2. Download the latest version for Windows.
3. When the installer opens, make sure you check the box at the bottom that says **"Add python.exe to PATH"**. If you skip this, your terminal won't know what Python is!
4. Click **Install Now**.

To verify it installed correctly, open your command prompt (cmd) and type:
```bash
python --version
pip --version
```
